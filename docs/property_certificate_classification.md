# 不动产权证书分类标准和字段定义

**日期**: 2026-07-03  
**目的**: 明确不动产权证书的分类标准和内容页字段提取范围

## 📋 分类标准

### 1. 附图页 (PROPERTY_CERTIFICATE_ATTACHMENT)

**特征关键词**：
- 附图
- 所在图幅编号
- 制图日期
- 制图者
- 宗地面积
- 宗地代码
- 房产分户图

**页面内容**：
- 房屋平面图
- 宗地图
- 图幅编号信息
- 制图相关信息

---

### 2. 首页 (PROPERTY_CERTIFICATE_FIRST_PAGE)

**特征关键词**：
- 登记机构（章）
- 编号（不动产权证号）
- 不动产编号

**页面内容**：
- 登记机构信息
- 证书编号
- 印章

**注意**：首页通常不包含具体的房屋信息，只有登记信息和编号。

---

### 3. 内容页 (PROPERTY_CERTIFICATE_CONTENT)

**特征关键词**（至少包含3个）：
- 不动产第 {数字}号
- 权利人
- 共有情况
- 坐落
- 不动产单元号
- 权利类型
- 权利性质
- 用途
- 面积
- 使用期限

**页面内容**：
- 不动产的核心信息
- 权利人信息
- 房屋详细信息
- 权利信息

## 📊 内容页字段定义（10个字段）

**只需要提取内容页中的数据项目**：

| 字段名 | 说明 | OCR文本示例 | 提取方式 |
|--------|------|------------|---------|
| **不动产编号** | 对应"不动产第 {数字}号" | "不动产第 1234567 号" | 正则：`不动产第\s*(\d+)\s*号` |
| **权利人** | 房屋所有权人姓名 | "权利人　张三" | 正则：`权利人[：:]\s*([^\n]+)` |
| **共有情况** | 单独所有/共同共有 | "共有情况　单独所有" | 正则：`共有情况[：:]\s*([^\n]+)` |
| **坐落** | 房屋坐落地址 | "坐落　蚌埠市蚌山区某某路XX号" | 正则：`坐落[：:]\s*([^\n]+)` |
| **不动产单元号** | 不动产唯一标识码 | "不动产单元号　340302100012GB00034F00012345" | 正则：`不动产单元号[：:]\s*([^\n]+)` |
| **权利类型** | 权利类型 | "权利类型　国有建设用地使用权/房屋所有权" | 正则：`权利类型[：:]\s*([^\n]+)` |
| **权利性质** | 权利性质 | "权利性质　出让/市场化商品房" | 正则：`权利性质[：:]\s*([^\n]+)` |
| **用途** | 房屋用途 | "用途　城镇住宅用地/住宅" | 正则：`用途[：:]\s*([^\n]+)` |
| **面积** | 房屋面积 | "面积　120.50平方米" | 正则：`面积[：:]\s*([\d.]+)` |
| **使用期限** | 使用期限 | "使用期限　2020年01月01日起至2070年12月31日止" | 正则：`使用期限[：:]\s*([^\n]+)` |

## 🔍 分类逻辑

```python
def classify_property_certificate(ocr_text: str) -> str:
    """判断不动产权证书的类型"""

    # 1. 附图页特征
    attachment_keywords = [
        "附图", "所在图幅编号", "制图日期", "制图者",
        "宗地面积", "宗地代码", "房产分户图"
    ]
    if any(kw in ocr_text for kw in attachment_keywords):
        return "PROPERTY_CERTIFICATE_ATTACHMENT"

    # 2. 首页特征（但不包含内容页特征）
    first_page_keywords = ["登记机构", "编号", "不动产权证号"]
    if any(kw in ocr_text for kw in first_page_keywords):
        if "不动产第" not in ocr_text and "权利人" not in ocr_text:
            return "PROPERTY_CERTIFICATE_FIRST_PAGE"

    # 3. 内容页特征（至少包含3个）
    content_keywords = [
        "不动产第", "权利人", "共有情况", "坐落",
        "不动产单元号", "权利类型", "权利性质", "用途", "面积", "使用期限"
    ]
    content_count = sum(1 for kw in content_keywords if kw in ocr_text)
    if content_count >= 3:
        return "PROPERTY_CERTIFICATE_CONTENT"

    # 4. 默认分类为内容页
    return "PROPERTY_CERTIFICATE_CONTENT"
```
