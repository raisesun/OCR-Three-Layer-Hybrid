## 📊 完成的工作

### 1. 图片旋转预处理分析

**分析结果**：
- 检查了可能需要旋转的图片（首页、附图页）
- EXIF方向都是1（正常方向），没有旋转问题
- OCR识别质量差是因为图片内容（图表、印章等），而非旋转

**结论**：
- ❌ 暂不添加旋转功能
- ✅ 保持当前预处理逻辑

### 3. 合同细分类别

**新增文档类型**：
- `PURCHASE_CONTRACT_FIRST_PAGE` - 购房合同首页
- `PURCHASE_CONTRACT_CONTENT` - 购房合同内容页
- `PURCHASE_CONTRACT_STAMP` - 购房合同签署页
- `STOCK_CONTRACT_FIRST_PAGE` - 存量房合同首页
- `STOCK_CONTRACT_CONTENT` - 存量房合同内容页
- `STOCK_CONTRACT_STAMP` - 存量房合同签署页

**分类规则**：

#### 购房合同/存量房合同

**首页特征**：
- 买卖合同、合同编号
- 卖方、买方、房屋坐落

**内容页特征**：
- 买卖合同字样
- 房屋基本情况、付款方式、装修价款

**签署页特征**：
- 合同签订日期、合同签订地址
- 签字、盖章、签章
- 可能有经纪机构字样

**注意**：需要与监管协议的签署页区分
- 监管协议签署页包含：资金监管、监管协议、监管凭证、监管资金

**路由规则**：
- 首页：VLM层
- 内容页：VLM层
- 签署页：规则层（返回空字段）

### 4. 公证书

**新增文档类型**：
- `NOTARY_CERTIFICATE` - 公证书

**分类特征**：
- 公证书（标题）
- 公证字第{数字}号（编号特征）

**处理规则**：
- 只需要分类，不需要提取字段
- 路由到规则层（返回空字段）

**处理规则**：
- 只需要分类，不需要提取字段

## 📋 分类规则总结

### 合同 vs 监管协议

| 特征 | 合同 | 监管协议 |
|------|------|---------|
| 标题 | 买卖合同 | 存量房交易资金监管协议 |
| 首页字段 | 卖方、买方、房屋坐落 | 编号、甲方、乙方、丙方 |
| 内容页特征 | 房屋基本情况、付款方式、装修价款 | 监管金额、监管账户 |
| 签署页特征 | 合同签订日期、合同签订地址 | 甲方（签章）、乙方（签章）、丙方（签章） |
| 区分关键 | 无"资金监管"字样 | 有"资金监管"、"监管协议"字样 |
