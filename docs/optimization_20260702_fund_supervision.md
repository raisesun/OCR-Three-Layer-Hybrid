# 资金监管业务文档优化 - 实施总结

**日期**: 2026-07-02  
**目标**: 扩展资金监管业务的文档类型，支持多页协议和凭证的分类与提取

## 1. 文档类型扩展

### 1.1 新增DocumentType

| 类型 | 说明 | 页面 |
|------|------|------|
| `FUND_SUPERVISION_AGREEMENT_FIRST_PAGE` | 资金监管协议首页 | 第1页 |
| `FUND_SUPERVISION_AGREEMENT_INFO_PAGE` | 资金监管协议信息页 | 第2页 |
| `FUND_SUPERVISION_AGREEMENT_STAMP` | 资金监管协议签章页 | 第3-4页 |
| `FUND_SUPERVISION_CERTIFICATE` | 资金监管凭证 | 单页 |

### 1.2 字段定义

#### 资金监管协议首页字段（13个）
- 编号、甲方、乙方、丙方
- 签署日期、网上签约备案合同号
- 房屋地址、建筑面积、不动产权证号
- 购房款(大写)、购房款(小写)
- 贷款(大写)、贷款(小写)

#### 资金监管协议信息页字段（8个）
- 甲方姓名、甲方身份证号、甲方银行、甲方账号
- 乙方姓名、乙方身份证号、乙方银行、乙方账号

#### 资金监管凭证字段（9个）
- 协议编号、日期
- 买房人、买房人姓名、身份证号
- 房屋坐落、建筑面积
- 监管总额、收款单位

## 2. 分类器优化

### 2.1 强信号检测

**资金监管凭证**:
- "存量房交易资金监管凭证"
- "资金监管凭证"

### 2.2 页面类型识别

| 页面 | 特征 | 结果 |
|------|------|------|
| 首页 | "存量房交易资金监管协议" + (编号/甲方/乙方/签署日期) | FIRST_PAGE |
| 信息页 | 身份证号/银行/账号 | PERSONAL_PAGE |
| 签章页 | 甲方（签章）/乙方（签章） | STAMP |

### 2.3 细化逻辑

```python
FUND_SUPERVISION:
  FIRST_PAGE → FUND_SUPERVISION_AGREEMENT_FIRST_PAGE
  PERSONAL_PAGE → FUND_SUPERVISION_AGREEMENT_INFO_PAGE
  STAMP → FUND_SUPERVISION_AGREEMENT_STAMP
```

## 3. 规则层优化

### 3.1 签章页处理

签章页直接返回空字段：
```python
FUND_SUPERVISION_AGREEMENT_STAMP → 返回空字段
```

### 3.2 首页提取逻辑

- **编号**: `[A-Z0-9\-]+`
- **甲乙丙方**: `甲方（卖方）：xxx` / `甲方：xxx`
- **签署日期**: `\d{4}年\d{1,2}月\d{1,2}日`
- **网上签约备案合同号**: `Y(2024)12345` 格式
- **房屋地址**: `房屋地址/坐落：xxx`
- **建筑面积**: `[\d.]+` + 平方米/㎡
- **不动产权证号**: `皖（2017）蚌埠市不动产权第XXXXXXX号`
- **购房款/贷款**: 大写金额 + 小写数字

### 3.3 信息页提取逻辑

- **甲方/乙方姓名**: `甲方姓名：xxx`
- **身份证号**: 18位身份证号
- **银行/账号**: 银行名称 + 账号

### 3.4 凭证提取逻辑

- **协议编号**: `[A-Z0-9\-]+`
- **日期**: `\d{4}年\d{1,2}月\d{1,2}日`
- **买房人信息**: 姓名 + 身份证号
- **房屋信息**: 坐落 + 面积
- **监管总额**: 数字金额
- **收款单位**: 公司名称（忽略红章）

## 4. VLM层优化

### 4.1 专用提示词

为每个新增文档类型创建了专用VLM提示词：
- `FUND_SUPERVISION_AGREEMENT_FIRST_PAGE`: 13个字段
- `FUND_SUPERVISION_AGREEMENT_INFO_PAGE`: 8个字段
- `FUND_SUPERVISION_CERTIFICATE`: 9个字段

### 4.2 基础类型映射

```python
FUND_SUPERVISION_AGREEMENT_FIRST_PAGE → FUND_SUPERVISION
FUND_SUPERVISION_AGREEMENT_INFO_PAGE → FUND_SUPERVISION
FUND_SUPERVISION_AGREEMENT_STAMP → FUND_SUPERVISION
FUND_SUPERVISION_CERTIFICATE → 独立Prompt
```

## 5. 路由配置

| 文档类型 | 路由 | 说明 |
|---------|------|------|
| FUND_SUPERVISION_AGREEMENT_FIRST_PAGE | RULE | 正则提取 |
| FUND_SUPERVISION_AGREEMENT_INFO_PAGE | RULE | 正则提取 |
| FUND_SUPERVISION_AGREEMENT_STAMP | RULE | 返回空字段 |
| FUND_SUPERVISION_CERTIFICATE | RULE | 正则提取 |
