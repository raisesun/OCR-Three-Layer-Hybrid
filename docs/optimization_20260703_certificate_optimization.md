## 🔧 优化内容

### 1. 日期字段 ✅

**问题**: 只支持"年月日"格式，实际凭证使用"YYYY-MM-DD"格式

**优化**:
```python
# 模式1: "年月日"格式
match = re.search(r'(\d{4}年\d{1,2}月\d{1,2}日)', full_text)
# 模式2: "YYYY-MM-DD"格式（凭证常用）
match = re.search(r'(\d{4}-\d{2}-\d{2})', full_text)
# 模式3: "YYYY/MM/DD"格式
match = re.search(r'(\d{4}/\d{2}/\d{2})', full_text)
```

**效果**: 成功提取 `2026-01-19`, `2026-01-16`

---

### 2. 买房人字段 ✅

**问题**: 凭证格式特殊，值在标签前面（"张新\n买房人"）

**优化**:
```python
# 模式1: "买房人：xxx"（标准格式）
match = re.search(r'买[房方]人\s*[:：]\s*([^\n]+)', full_text)
# 模式2: "xxx\n买房人"（凭证格式，值在标签前面）
match = re.search(r'([一-龥]{2,4})\s*\n\s*买房人', full_text)
```

**效果**: 成功提取 `张新`, `胡海军`

---

### 3. 身份证号字段 ✅

**问题**: 提取到卖房人身份证号而非买房人

**优化**:
```python
# 找到所有身份证号
matches = re.findall(r'(\d{17}[\dXx])', full_text)
# 凭证格式：第一个是卖房人，第二个是买房人
if len(matches) >= 2:
    fields["身份证号"] = matches[1].upper()  # 取第二个（买房人）
else:
    fields["身份证号"] = matches[0].upper()
```

**效果**: 
- 优化前: `340311198508051213` (卖房人)
- 优化后: `34032119830616153X` (买房人) ✓

### 4. 协议编号字段 ✅

**问题**: 只支持标签在前的格式

**优化**:
```python
# 模式1: "协议编号：xxx"
match = re.search(r'(?:协议编号|编\s*号)\s*[:：]?\s*([A-Z0-9\-]+)', full_text)
# 模式2: "协议编号\nxxx"（标签在前）
match = re.search(r'协议编号\s*\n\s*([A-Z0-9\-]+)', full_text)
# 模式3: "xxx\n协议编号"（值在前）
match = re.search(r'([A-Z0-9\-]+)\s*\n\s*协议编号', full_text)
```

**效果**: 成功提取 `2026011900010627`, `2026011600010591`

### 5. 房屋坐落字段 ✅

**问题**: 提取到金额而非地址

**优化**:
```python
# 模式1: "房屋坐落：xxx"
match = re.search(r'(?:房屋坐落|坐落|房屋地址)\s*[:：]\s*([^\n]+)', full_text)
# 模式2: "房屋坐落\nxxx"（标签在前）
match = re.search(r'(?:房屋坐落|坐落)\s*\n\s*([^\n]+)', full_text)
# 排除金额
if match and re.match(r'[零壹贰叁肆伍陆柒捌拾拾佰仟万亿元整]+', match.group(1).strip()):
    match = None
# 模式3: "xxx\n房屋坐落"（值在前）
match = re.search(r'([^\n]*(?:号|室|栋|楼|单元|层|座|幢))\s*\n\s*(?:房屋坐落|坐落)', full_text)
```

**效果**: 
- 优化前: `贰拾壹万捌仟元整` (金额) ❌
- 优化后: `琥珀花园27号楼3单元6层602号` (地址) ✓

### 6. 建筑面积字段 ✅

**问题**: 表格布局导致值在标签前面，中间有其他字段

**优化**:
```python
# 模式1-3: 标准格式和简单前后格式
# 模式4: 找到"建筑面积"，然后往前找最近的数字+单位
lines = full_text.split('\n')
for i, line in enumerate(lines):
    if '建筑面积' in line:
        # 往前搜索最多5行
        for j in range(max(0, i-5), i):
            area_match = re.search(r'([\d.]+)\s*(?:平方米|㎡|m2|m²)', lines[j])
            if area_match:
                fields["建筑面积"] = area_match.group(1)
                break
```

**效果**: 成功提取 `92.23`, `84.62`

## 🎯 关键发现

### 1. 凭证格式特殊性

凭证采用**表格布局**，标签可能在值前面或后面：

**格式1**（标签在前）:
```
建筑面积
92.23 m²
```

**格式2**（值在前）:
```
张新
买房人
```

**格式3**（值在前，中间有其他字段）:
```
84.62 m²
资金余额
建筑面积
```

### 2. OCR读取顺序

OCR按行读取，但表格布局导致值的顺序不固定，需要灵活处理前后关系。

### 3. 分类器问题

部分"凭证"实际上是协议信息页（包含银行账号信息），需要改进分类器。
