### 问题3：分类逻辑优化

**已修复的问题**：
1. ✅ 添加了文档类型预检查（排除资金监管凭证等非不动产权证书文档）
2. ✅ 调整了内容页特征的优先级（优先于附图页）
3. ✅ 移除了"面积"作为附图页特征（太宽泛，会误匹配"共有宗地面积"）

**优化后的分类逻辑**：
```python
# 0. 首先检查是否是不动产权证书
if "不动产权" not in ocr_text and "不动产第" not in ocr_text:
    # 检查是否是资金监管凭证
    if "资金监管凭证" in ocr_text:
        return "FUND_SUPERVISION_CERTIFICATE"
    return "UNKNOWN"

# 1. 内容页特征（优先级最高）
content_keywords = ["不动产第", "权利人", "共有情况", "坐落", ...]
if content_count >= 3:
    return "PROPERTY_CERTIFICATE_CONTENT"

# 2. 附图页特征
attachment_keywords = ["附图", "所在图幅编号", "制图日期", ...]
if any(kw in ocr_text):
    return "PROPERTY_CERTIFICATE_ATTACHMENT"

# 3. 首页特征
if "登记机构" in ocr_text and "权利人" not in ocr_text:
    return "PROPERTY_CERTIFICATE_FIRST_PAGE"

# 4. 默认
return "PROPERTY_CERTIFICATE_CONTENT"
```
