## 二、正确配置详解

### 2.1 完整配置代码

```python
from paddleocr import PPChatOCRv4Doc

pp_chatocr = PPChatOCRv4Doc(
    # 启用所有OCR功能（必需！）
    use_doc_orientation_classify=True,   # 文档方向分类
    use_doc_unwarping=True,              # 文档矫正
    use_textline_orientation=True,       # 文本行方向
    use_seal_recognition=True,           # 印章识别
    use_table_recognition=True,          # 表格识别

    # LLM配置（使用官方推荐的qwen2.5:1.5b）
    chat_bot_config={
        "api_type": "openai",
        "model_name": "qwen2.5:1.5b",
        "base_url": "http://localhost:11434/v1",
        "api_key": "ollama"
    },

    # 向量检索配置（必需！）
    retriever_config={
        "api_type": "openai",
        "model_name": "nomic-embed-text",
        "base_url": "http://localhost:11434/v1",
        "api_key": "ollama"
    }
)
```

### 2.2 必需的模型

| 模型 | 用途 | 大小 | Ollama命令 |
|------|------|------|-----------|
| qwen2.5:1.5b | LLM | 986MB | `ollama pull qwen2.5:1.5b` |
| nomic-embed-text | 向量检索 | 274MB | `ollama pull nomic-embed-text` |

### 2.3 正确的API调用

```python
# 步骤1：视觉信息提取
visual_result = pp_chatocr.visual_predict(
    input=image_path,
    use_doc_orientation_classify=True,
    use_doc_unwarping=True,
    use_textline_orientation=True,
    use_seal_recognition=True,
    use_table_recognition=True,
)

visual_info = visual_result[0]['visual_info']

# 步骤2：LLM信息提取
result = pp_chatocr.chat(
    key_list=["买受人", "出卖人", "合同编号"],
    visual_info=visual_info
)
```
