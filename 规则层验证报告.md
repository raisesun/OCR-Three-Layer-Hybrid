## 五、优化建议

### 5.1 短期优化（本周）

#### 1. 优化正则表达式

**身份证正面：**
```python
# 当前正则（过于严格）
name_match = re.search(r'姓名\s*([一-龥]{2,4})', full_text)

# 优化后（更灵活）
name_patterns = [
    r'姓名\s*([一-龥]{2,4})',
    r'姓名([一-龥]{2,4})',
    r'([一-龥]{2,4})\s*男\s*[一-龥]+\s*出生',
]
for pattern in name_patterns:
    match = re.search(pattern, full_text)
    if match:
        fields['姓名'] = match.group(1)
        break
```

**结婚证：**
```python
# 增加更多格式匹配

]
```

